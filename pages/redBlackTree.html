Binary Search Tree Enhancement - DSA Artifact

This artifact is a conversion of the binary search tree in C++ to a red-black-tree in C#. Itâ€™s a self-balancing BST that was created in 1972, 10 years after the initial creation of the AVL tree, another type of self-balancing tree. I used this artifact since it requires several enhancements on top of a language overhaul.

Generics

The first enhancements over the initial C++ implementation are a generic type implementation with the IObservable interface requiring all values to implement the CompareTo() method for tree traversal.

Tree Balancing

The largest challenges had to do with understanding how the new tree methods work. The rotations were slightly confusing, but it took videos and visualization tools to understand transplants for deletions. I ended up changing some details. I ended up ditching the ref type for simplicity, although in retrospect it seems to operate like a pointer. I also swapped a color from an enum to a bool since there are only two possible values, red or not red, and made a few more adjustments.
